# My E-Portfolio

Personal portfolio website for Jess Anthony M. Tahil — showcasing background, skills, and projects. Built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it locally.

## Stack

- [Next.js](https://nextjs.org) (App Router, Turbopack)
- Tailwind CSS v4
- Framer Motion
- React Three Fiber (particle background)

## Deploy

Deployed on [Vercel](https://vercel.com/new?utm_medium=default-template&filter=next.js&utm_source=create-next-app&utm_campaign=create-next-app-readme).
